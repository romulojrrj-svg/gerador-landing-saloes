# Salon Landing Generator

Initial Next.js project for registering beauty salons and previewing generated landing pages.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock data only, with no database and no AI API yet

## Routes

- `/` - product home
- `/salons/new` - visual salon intake form
- `/salons/[id]/preview` - mocked landing page preview

## Run

```bash
npm install
npm run dev
```
